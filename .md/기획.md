## 6. 연관 페이지 그룹 (흐름)

### A. Story 작성 흐름

```
/app (QuestionCard)
  → /app/write (create)
  → [저장] → /app/drawer 또는 /app
/app/drawer/[id]
  → /app/write/[id] (edit)
```

- **공개 선택 시:** `emailVerified` gate → 미인증 toast/sheet
- **캡슐:** write에서 CapsuleDialog → capsule API 또는 story+capsule flag
- **mutate:** stories, drawer, public list

### B. Public Story 소셜

```
/app (PublicStoriesFeed)
  → /app/public/[id]
  → 댓글·반응·신고
```

- CommunityPost와 **별도** — Story `visibility=community`
- SEO: sitemap + `generateMetadata`

### C. Community 토론

```
/app/community (feed + search)
  → /app/community/write
  → /app/community/[id] (댓글·멘션·알림)
  → /app/community/write/[id] (수정)
/app/notifications → /app/community/[id]
```

- **이메일 인증** 필수 (write, comment)
- 알림은 CommunityPost 기준만

### D. Capsule

```
/app/capsule
  → /app/capsule/write → [봉인] → /app/capsule
  → /app/capsule/[id] (봉인/해제)
/app/drawer (capsule banner → /app/capsule)
```

### E. Auth · Settings

```
최초 접속: DeviceIdBootstrap → POST /users/guest
/app/settings
  → login / signup → merge guest
  → delete account
```

### F. 서랍 탐색

```
/app/drawer
  → 잔디 날짜 선택 / 검색 / 북마크 필터
  → /app/drawer/[id]
  → DrawerRevisitCard (랜덤 과거 글)
```

| 타임캡슐    | 봉인·해제, 개수 무제한, 푸시 ⏳                                    |
| 커뮤니티    | **공개 Story**(홈) + **토론**(CommunityPost) 이원화, 신고 3건 숨김 |
| 계정        | 게스트 device_id, community·댓글은 **이메일 인증**                 |
| 수익        | F18 배너 placeholder, F19 IAP ⏳                                   |

## 8. 사용자 흐름

### 최초 실행 (구현)

```
접속 → device_id(localStorage) → POST /users/guest → /app (온보딩 없음)
```

### 매일

```
/app → [이야기 쓰기] → /app/write → 저장 → drawer
/app → 공개 feed → /app/public/[id]
```

### 회원 전환

```
/settings → signup/login → guest merge → member
```

### 1년 Echo (부분)

```
서랍·상세에서 Echo 뱃지 표시 → P5 비교 화면 없음
```
